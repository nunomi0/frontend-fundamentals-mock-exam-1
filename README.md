# Frontend Fundamentals 모의고사

## Getting started

```sh
yarn dev
```

## 기능
- [x] 적금 상품 목록 연동하기
    - [x] GET /api/savings-products
    - [x] 콤마 출력 확인
- [x] 저축 목표 입력 기능 만들기
    - [x] 목표 금액, 월 납입액, 저축 기간 입력
    - [x] 적금 상품 목록 필터링
- [ ] 적금 상품 목록에서 선택 기능 만들기
- [ ] 계산 결과 탭 만들기
    - [ ] CalculationResult 컴포넌트 표시
    - [ ] 예상 수익 금액, 목표 금액과의 차이, 추천 월 납입 금액 계산
    - [ ] 적금 상품 선택하지 않은 경우 “상품을 선택해주세요”를 출력
- [ ] 추천 상품 목록 만들기
    - [ ] 조건에 맞는 이자율 가장 높은 2개 상품 출력
    - [ ] 선택한 상품은 아이콘 표기
